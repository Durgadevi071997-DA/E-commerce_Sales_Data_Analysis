
# E-Commerce Sales Data Analysis




## Introduction

The E-commerce Sales Data Analysis project focuses on analyzing online sales data to understand customer purchasing behavior, product performance, and regional sales trends. The project uses data visualization and analytical techniques to generate meaningful business insights for better decision-making.
## Objectives

Analyze overall sales performance

Identify top-selling products and categories

Understand customer buying patterns

Compare regional sales performance

Improve business strategies using data insights
## Tools Used

Microsoft Excel

Power BI

Data Cleaning

Data Visualization

Dashboard Creation
## Key Insights

Certain product categories generated higher sales and profits

Some regions showed better sales performance than others

Seasonal trends affected customer purchasing behavior

Top customers contributed significantly to total revenue

Interactive dashboards improved data understanding
## Key Performance Indicators (KPI)

Total Sales

Total Profit

Quantity Sold

Average Order Value

Regional Sales Performance

Top-Selling Products

Customer Count
